Flagged Comment Moderation

System administrators have access to moderate comment which have been flagged.

To moderate comment, go to: MORE > ADMIN PORTAL.  Then within the admin portal, MODERATION > FLAGGED COMMENTS.

Comments displayed are restricted to users within the administrators defined territory.  A global admin will see comments flagged from all comment authors.  A territory admin will only see comments flagged from authors in their territory.

The page will display the following fields:

  • Author - The user who wrote the comment
  • Title - The title of the learning object or playlist where the comment was entered
  • Comment - The comment itself
  • Date created - The date the comment was entered

 

Taking action on a comment

By clicking on the row of the comment a Reported Comment modal will open.  The admin will have two options:

  • Remove Comment.  This will both remove the comment and the flagged comment message from the respective details page.  This action will also reduce the comment count for the item the comment was found on.
  • Allow comment.  This will remove the flagged comment message and restore the comment to the respective details page.
