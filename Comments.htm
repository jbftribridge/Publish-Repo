Comments

The Comments feature enables users to engage with the content by leaving direct feedback as one method of providing feedback to the content owner.  Users can view the volume of comments on an item from both tiles and the details pages.

Viewing the comments

Clicking the comment icon on a tile or the comment count on the details page navigates the user to the comment section within the details page.  comments are sorted oldest to newest by default.

 

Add a comment

Click the Add a comment button to open the dialog box.  Once you enter characters the Save button is activated. Click Save to save the comment and it is added to the bottom of the page.

Delete a comment

A Delete option is available next to any comment that the user themselves has added in the system.

Flag a comment

A Flag option is available next to any comment that other users have left in the system.  A comment should be flagged if it is not suitable as feedback for the content owner.   If flagged the comment itself is replaced by a message that the comment has been flagged.  Flagged comments are reviewable and can either be removed or reistated by administrators.  To learn more about the comment moderation: See Comment Moderation.
