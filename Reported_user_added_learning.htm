Reported user added learning

System administrators have access to moderate user added content which has been reported.

To moderate comment, go to: MORE > ADMIN PORTAL.  Then within the admin portal, MODERATION > REPORTED USER ADDED LEARNING.

Content displayed are restricted to users within the administrators defined territiory.  A global admin will see content reported from all authors.  A territory admin will only see content reported by authors in their territory.

The page will display the following fields:

  • Owner - The user who created the user added learning
  • Title - The title of the user added learning
  • Group - The territory of the user who created the content
  • Date reported - The date in which the content was reported

Taking action on a reported user added learning

NOTE: Flagged user added content is marked as inactive on the playlist which it was found.

By clicking on the row of the item a Reported learning item modal will open.  The admin will have two options:

  • Remove learning item.  This will remove the item from the moderation report.  The learning item will remain as inactive in the system.
  • Allow learning item.  This will re-activate the user added learning in the system.
